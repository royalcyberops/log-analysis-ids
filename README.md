# Log Analysis & Intrusion Detection Tool

A cybersecurity portfolio project that analyzes logs, detects suspicious activity, and generates security reports.

## Features (MVP)
- Failed login detection
- Brute-force attack detection
- Suspicious IP detection
- JSON security reports
